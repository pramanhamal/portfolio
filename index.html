Artifact 2: Neural Network Components Visualization

This portfolio artifact explains the structure and function of neural networks in a clear visual format. It highlights the major components that allow neural networks to process information, learn patterns, reduce prediction errors, and improve performance over time.

Neural Network Architecture

A neural network is organized into an input layer, one or more hidden layers, and an output layer. Data flows from the input layer through hidden layers where calculations and transformations occur. The output layer then produces the final prediction or classification.

Key Components

  • Layers: Groups of neurons that organize the flow of information. Input layers receive data, hidden layers process patterns, and output layers produce results.
  • Neurons: Computational units that receive inputs, apply weights, perform calculations, and pass outputs forward.
  • Weights: Parameters that determine how strongly inputs influence neuron outputs. Training adjusts weights to improve accuracy.
  • Activation Functions: Functions such as ReLU, Sigmoid, and Tanh that determine whether neurons activate and pass information forward.
  • Loss Functions: Measurements of prediction error. Lower loss means the model is performing better.
  • Optimization Algorithms: Methods such as Gradient Descent and Adam that adjust weights to reduce loss and improve predictions.

Neural Network Playground Insights

Experimenting with Neural Network Playground showed that increasing hidden layers can help a model learn more complex patterns, but it can also increase training time. Adding noise made prediction more difficult and reduced accuracy. Feature selection also had a major impact on model performance because better input features helped the network separate patterns more clearly.
